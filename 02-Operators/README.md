# Java Operators

This folder demonstrates the different types of operators available in Java.

## Programs Included

1. Arithmetic Operators
2. Assignment Operators
3. Relational Operators
4. Logical Operators
5. Unary Operators
6. Bitwise Operators
7. Ternary Operator

## Operators Covered

| Operator Type | Symbols |
|---------------|---------|
| Arithmetic | + - * / % |
| Assignment | = += -= *= /= %= |
| Relational | == != > < >= <= |
| Logical | && || ! |
| Unary | ++ -- + - |
| Bitwise | & \| ^ ~ << >> |
| Ternary | ? : |

## Learning Outcomes

After completing this folder, you will be able to:

- Perform arithmetic calculations.
- Compare values using relational operators.
- Use logical operators for decision making.
- Understand pre/post increment and decrement.
- Work with bitwise operations.
- Write concise conditional expressions using the ternary operator.
